## Make the plot

Next, we want to know how does Delta 15 N (o/oo) and Delta 13 C (o/oo) affect the Species. To visualize this relationship, we create a scatterplot using matplotlib and seaborn:d


```python
from matplotlib import pyplot as plt
import seaborn as sns
fgrid = sns.relplot(x = "Delta 15 N (o/oo)", 
                    y = "Delta 13 C (o/oo)", 
                    hue = "Species", 
                    data = penguins).set(title = "My Awesome Penguins Plot")
# create the scatterplot with the x-axis represents Delta 15 N (o/oo), and y-axis represents Delta 13 C (o/oo). 
# the scatterplot use different color to reprsents different penguins species.
```
